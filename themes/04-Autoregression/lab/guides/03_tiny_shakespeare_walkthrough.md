# Пошаговый разбор: `04-Autoregression / ЛР02 CPU` на `Tiny Shakespeare`

## Канонический порядок
1. Завершить `ЛР01`.
2. Открыть `02_decoder_only_tiny_shakespeare.ipynb`.
3. Выполнять `TODO` строго от 1 до 6.
4. После каждого `TODO` запускать мини-проверку.
5. При затруднениях использовать `04_tiny_shakespeare_debugging_playbook.md`.

## Зачем нужен отдельный CPU-ноутбук
CPU-вариант фиксирует минимально обязательный прикладной перенос:
- реальная статистика текста;
- детерминированный маршрут;
- воспроизводимые метрики и генерация.

Это базовая контрольная точка перед масштабированием в `GPU`-варианте.

## Критерии прохождения
- `test_perplexity < baseline_perplexity`;
- `success_count >= cfg['gen_threshold']`;
- `mean_match_ratio >= cfg['gen_mean_threshold']`;
- диагностика внимания проходит.
